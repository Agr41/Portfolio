Pokemon Web

Mi sitio web es una plataforma dinámica impulsada por los servicios de alojamiento de Firebase, que cuenta con opciones de inicio de sesión convenientes a través de Facebook, Google y Git. El desarrollo frontend está hábilmente diseñado utilizando HTML, CSS y JavaScript, garantizando una experiencia de usuario visualmente atractiva y fluida. Uno de los aspectos destacados de mi sitio web es su integración con una API de Pokémon. Tendrás acceso a una extensa base de datos de Pokémon, con información detallada sobre sus estadísticas, habilidades, evoluciones y conjuntos de movimientos. He añadido una función que te permite diseñar tu propio Pokémon. Puedes guardar, editar e incluso eliminar tus creaciones personalizadas, dándote control total sobre su apariencia, habilidades y características. ¡Entra y crea tu propio Pokémon!
